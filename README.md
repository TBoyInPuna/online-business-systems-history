# Online Business Systems (San Francisco)

This repository contains a technical and historical recollection of
Online Business Systems, a San Francisco IBM mainframe service bureau,
covering the period 1973–1982.

It documents the OBS CICS system, WYLBUR/MILTEN, and related work.
